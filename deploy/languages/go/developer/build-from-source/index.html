Prerequisites

Section titled “Prerequisites”
  • Go
  • GNU Make
  • git
  • GCC
  • pkg-config
  • cmake
  • protoc (protobuf compiler) >= v3.20.0
  • openssl
  • openssl-dev
  • rustup
  • ziglang and zigbuild (for GNU Linux only)
  • valkey (for testing)
