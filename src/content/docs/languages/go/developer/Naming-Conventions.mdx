## Naming Conventions

### Function names

For every command that you implement, please use PascalCase.

Examples of good command function names:

* `BZPopMin`
* `ZRem`
* `PExpireWithOptions`
* `SetWithOptions`
* `Decr`

Examples of bad command function names:

* `zRange`
* `hincrbyfloat`
* `Sdiffstore`
