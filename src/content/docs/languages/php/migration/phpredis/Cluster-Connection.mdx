## PHPRedis
```php
$redis = new RedisCluster(null, ['127.0.0.1:7000', '127.0.0.1:7001']);
```

## Valkey GLIDE PHP
```php
use ValkeyGlide\ValkeyGlideCluster;

$client = new ValkeyGlideCluster([
    ['host' => '127.0.0.1', 'port' => 7000],
    ['host' => '127.0.0.1', 'port' => 7001]
]);
```
