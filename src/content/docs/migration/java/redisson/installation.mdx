## Quick Installation

Add the following dependency to your `pom.xml` file:

<Code
  lang="xml"
  code={`
<dependency>
<groupId>io.valkey</groupId>
<artifactId>valkey-glide</artifactId>
<version>${javaGlideLatest}</version>
</dependency>
`}
/>

## Full Installations

This section includes setup instructions for Gradle, Maven, and SBT, with or without OS detection tools.

:::tip[Uber Jar vs Classifier]
Starting from version 2.3.0, you can use the uber jar (no classifier needed), which bundles all platform binaries.
The uber jar is larger in size. If you prefer a smaller dependency, use a platform-specific classifier for your OS.
:::

<Tabs>
  <TabItem label="Gradle">
    <Aside>must include a `classifier` to specify your platform.</Aside>

    Example with osdector:

    ```groovy
    // with osdetector
    plugins {
        id "com.google.osdetector" version "1.7.3"
    }
    dependencies {
        implementation group: 'io.valkey', name: 'valkey-glide', version: '2.+', classifier: osdetector.classifier
    }
    ```

    Example with uber jar:

    ```groovy
    // with uber jar
    dependencies {
        implementation group: 'io.valkey', name: 'valkey-glide', version: '2.+'
    }
    ```

    Example with osx-aarch\_64:

    ```groovy
    // osx-aarch_64
    dependencies {
        implementation group: 'io.valkey', name: 'valkey-glide', version: '2.+', classifier: 'osx-aarch_64'
    }
    ```

    Example with osx-x86\_64:

    ```groovy
    // osx-x86_64
    dependencies {
        implementation group: 'io.valkey', name: 'valkey-glide', version: '2.+', classifier: 'osx-x86_64'
    }
    ```

    Example with linux-aarch\_64:

    ```groovy
    // linux-aarch_64
    dependencies {
        implementation group: 'io.valkey', name: 'valkey-glide', version: '2.+', classifier: 'linux-aarch_64'
    }
    ```

    Example with linux-x86\_64:

    ```groovy
    // linux-x86_64
    dependencies {
        implementation group: 'io.valkey', name: 'valkey-glide', version: '2.+', classifier: 'linux-x86_64'
    }
    ```

    Example with windows-x86\_64:

    ```groovy
    // windows-x86_64
    dependencies {
        implementation group: 'io.valkey', name: 'valkey-glide', version: '2.+', classifier: 'windows-x86_64'
    }
    ```
  </TabItem>

  <TabItem label="Maven">
    <Aside>
      Must include a `classifier`.
    </Aside>

    <Aside>
      Please use this dependency block, or both the dependency and the extension blocks if you're using `os-maven-plugin`, and add it to the `pom.xml` file.
    </Aside>

    <Code
      lang="xml"
      code={`
<!--uber jar-->
<dependency>
<groupId>io.valkey</groupId>
<artifactId>valkey-glide</artifactId>
<version>${javaGlideLatest}</version>
</dependency>

<!--osx-aarch_64-->
<dependency>
<groupId>io.valkey</groupId>
<artifactId>valkey-glide</artifactId>
<classifier>osx-aarch_64</classifier>
<version>${javaGlideLatest}</version>
</dependency>

<!--linux-aarch_64-->
<dependency>
<groupId>io.valkey</groupId>
<artifactId>valkey-glide</artifactId>
<classifier>linux-aarch_64</classifier>
<version>${javaGlideLatest}</version>
</dependency>

<!--linux-x86_64-->
<dependency>
<groupId>io.valkey</groupId>
<artifactId>valkey-glide</artifactId>
<classifier>linux-x86_64</classifier>
<version>${javaGlideLatest}</version>
</dependency>

<!--with os-maven-plugin-->
<build>
<extensions>
<extension>
<groupId>kr.motd.maven</groupId>
<artifactId>os-maven-plugin</artifactId>
</extension>
</extensions>
</build>
<dependencies>
<dependency>
<groupId>io.valkey</groupId>
<artifactId>valkey-glide</artifactId>
<classifier>\${os.detected.classifier}</classifier>
<version>${javaGlideLatest}</version>
</dependency>
</dependencies>`}
    />
  </TabItem>

  <TabItem label="SBT">
    <Aside>
      must include a `classifier`. Please use this dependency block and add it to the build.sbt file.
    </Aside>

    ```scala
    // uber jar
    libraryDependencies += "io.valkey" % "valkey-glide" % "2.+"

    // osx-aarch_64
    libraryDependencies += "io.valkey" % "valkey-glide" % "2.+" classifier "osx-aarch_64"

    // linux-aarch_64
    libraryDependencies += "io.valkey" % "valkey-glide" % "2.+" classifier "linux-aarch_64"

    // linux-x86_64
    libraryDependencies += "io.valkey" % "valkey-glide" % "2.+" classifier "linux-x86_64"
    ```
  </TabItem>
